1-*RAMDEKİ TABLE'DAKİ İSTEDİĞİM KOLONLARI İNTERNAL TYPEA ATIP HEPSİNE TYPE VERİP SONRA BUNLARI İNTERNAL LİST'E ATIYORUM

TYPES: BEGIN OF gty_list,
  CARRID TYPE S_CARR_ID,
  CARRNAME TYPE S_CARRNAME,
  FLDATE TYPE S_DATE,
  CONNID TYPE S_CONN_ID,
    END OF gty_list.

  DATA: gt_list TYPE TABLE OF gty_list,
        gs_list TYPE gty_list.

3-FİELD CATOLOG OLUŞTURUYORUZ. (ilk type'ı fonksiyon içinden structeri ise internal table oluşturduğum type'dan)

DATA: gt_fieldcatolog TYPE SLIS_T_FIELDCAT_ALV,
      gs_fieldcatolog TYPE slis_fieldcat_alv.



START-OF-SELECTION.


2-*BURAYA DA INTERNAL TABLE'A SORGULARI ATIP VERİ ALMASINI SAĞLIYORUM

SELECT
 SCARR~CARRID
 SCARR~CARRNAME
 SFLIGHT~FLDATE
 SFLIGHT~CONNID
  FROM SCARR
  INNER JOIN SFLIGHT on SFLIGHT~CARRID eq SCARR~CARRID
  INTO TABLE gt_list.

4- FİELD CATOLOGDA FİELD NAME OLUŞTURUP FİELD NAME'E KOLONU VERİP BUNU APPEND İLE GT_FİELDCATOLOG'A ATIYORUZ ALVDE GÖSTERMESİ İÇİN. (BÜTÜN KOLONLARI TEK TEK YAPIYORUZ)

CLEAR: gs_fieldcatolog.
  gs_fieldcatolog-fieldname = 'CARRID'.
-------------------------
  APPEND gs_fieldcatolog TO gt_fieldcatolog.

  CLEAR: gs_fieldcatolog.
  gs_fieldcatolog-fieldname = 'CARRNAME'.
----------------------
  APPEND gs_fieldcatolog TO gt_fieldcatolog.

CLEAR: gs_fieldcatolog.
  gs_fieldcatolog-fieldname = 'FLDATE'.
---------------------
 APPEND gs_fieldcatolog TO gt_fieldcatolog.

CLEAR: gs_fieldcatolog.
  gs_fieldcatolog-fieldname = 'CONNID'.
---------------------
  APPEND gs_fieldcatolog TO gt_fieldcatolog.
